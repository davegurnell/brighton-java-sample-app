    Rough-and-ready replacement for java.io.Console that provides easy access to a readLine from stdin.

    This class exists because java.io.Console doesn't work well with the Eclipse REPL.

    public class Console {

  private static BufferedReader in = new BufferedReader(new InputStreamReader(System.in));

      public static String readLine(String prompt) {
    try {
      out.print(prompt);
      return in.readLine();
    } catch (IOException e) {
      throw new RuntimeException(e);
    }
